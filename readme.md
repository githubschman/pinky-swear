# Pinky Swear

a small node project for exploring Promise functionality added with ES6

---

## Starting Up
1. pinky swear
![](https://media.giphy.com/media/l3UcmIFKn6dCvpYyI/giphy.gif)

2. install dependencies
```yarn```
or
```npm install```

3. run npm start to start the program
```npm start```

4. promise
